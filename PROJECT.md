## How It Works

Modern face recognition uses **embeddings** — a ~512-dimensional vector that uniquely fingerprints a face, generated by a neural network. Comparing two faces = comparing two vectors (cosine similarity). If similarity > threshold → match.

**Workflow:**
```
inbound photo
  → extract face(s)
  → generate embedding(s)
  → compare against people/ database
  → if match: "That's Jason (confidence: 0.94)"
  → if no match: "Unknown person"
  → inject matched names into session context automatically
```

## Recommended Library: `face_recognition`

- Built on dlib's deep learning model
- ~150MB model weights
- Extremely simple API — 1-2 lines to compare faces
- Accurate enough for personal use (1:N matching)
- Python, runs locally, no cloud dependency

```python
import face_recognition
import numpy as np

# Build reference
known_img = face_recognition.load_image_file("people/jason.jpg")
known_enc = face_recognition.face_encodings(known_img)[0]
np.save("people/jason.npy", known_enc)

# Identify in new photo
unknown_img = face_recognition.load_image_file("photo.jpg")
unknown_encs = face_recognition.face_encodings(unknown_img)
for enc in unknown_encs:
    matches = face_recognition.compare_faces([known_enc], enc, tolerance=0.6)
    dist = face_recognition.face_distance([known_enc], enc)[0]
    confidence = 1 - dist
    if matches[0]:
        print(f"Jason — confidence: {confidence:.2f}")
```

## Scripts to Build

### `scripts/build_people_db.py`
- Scans `workspace/people/*.jpg`
- Generates `.npy` embedding for each
- Reports any photos where no face was detected

### `scripts/identify_faces.py`
- Input: image path
- Loads all `.npy` embeddings from `people/`
- Detects faces in image, compares each against all known embeddings
- Returns: list of `{name, confidence, location}` for each face found
- Unknown faces reported as `Unknown (no match)`

### Integration with inbound photos
- Hook into `process_attachment.py` (already exists)
- After downloading an attachment, run `identify_faces.py`
- Inject results into session context: *"Faces detected: Jason Cox (0.94), James Tatum (0.89)"*
